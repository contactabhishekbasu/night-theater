# Night Theater — citizen-science exoplanet pipeline (public site)

Static snapshots of a fully automated exoplanet transit-reduction pipeline
(MicroObservatory / ExoWorlds / LCO public data → EXOTIC → red-noise-honest
timing) with a human review gate before any submission.

- **/status/** — pipeline mission control (weekly snapshot)
- **/review/** — peer-review-grade per-run dossiers (figures, uncertainties, provenance)
- **/eval/** — model evaluation: frozen protocol, baselines, calibration, lineage

Data refreshes weekly via automation. Source pipeline is private; methods are
documented in the dossiers and evaluation page.
